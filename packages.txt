  Net-SNMP
  https://packages.debian.org/wheezy/amd64/snmp/filelist

Binaries in snmp package
========================
snmpget        - communicates with a network entity using SNMP GET requests

snmpgetnext    - communicates with  a network entity using SNMP GETNEXT
                 requests 

snmpset        - communicates with a network entity using SNMP SET requests

snmptranslate  - translate  MIB  OID names between numeric and
                 textual forms

snmpwalk       - retrieve a subtree of management values using SNMP GETNEXT
                 requests
